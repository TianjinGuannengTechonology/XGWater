# XGWater

一个记录喝水情况并有提醒功能的小应用

可以查看本周与上周喝水量的对比，和总的喝水记录。在设置界面可以更改一些设置

该应用还有提醒功能，每隔一段时间都会提醒用户喝水。

![](https://github.com/TianjinGuannengTechonology/XGWater/blob/master/bg.png)

### Framework
Center center server

Dbproxy database agent

Gate gateway server

Hub micro service node
